# Chip Hero: Campus Adventure - Phase 3 Crystals and Score

This MakeCode Arcade project extends the Phase 2 minimal platformer with energy crystals and scoring.

Included features:

- Original player sprite.
- Simple horizontal tilemap.
- Player start position.
- Left and right movement.
- Gravity.
- Ground-only jump.
- Camera follow.
- Goal portal.
- Win screen on goal overlap.
- Failure screen when the player falls below the map.
- Crystal SpriteKind.
- Four fixed energy crystals.
- Score starts at 0.
- Player touching a crystal adds 1 point.
- Collected crystals are destroyed immediately.
- Short collection sound.

Not included in this phase:

- Coins.
- Enemies.
- Life.
- Second level.
- Boss.
- Shooting.
- Moving platforms.
- Random crystal generation.
- Chest.
- Extra extensions.
