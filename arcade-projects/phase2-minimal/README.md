# Chip Hero: Campus Adventure - Phase 2 Minimal

This MakeCode Arcade project is the minimal playable platformer for Phase 2.

Included features:

- Original player sprite.
- Simple horizontal tilemap.
- Player start position.
- Left and right movement.
- Gravity.
- Ground-only jump.
- Camera follow.
- Goal portal.
- Win screen on goal overlap.
- Failure screen when the player falls below the map.

Not included in this phase:

- Coins.
- Enemies.
- Life.
- Second level.
- Boss.
- Shooting.
- Moving platforms.
